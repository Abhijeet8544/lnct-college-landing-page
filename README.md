# lnct-college-landing-page
LNCT College Landing Page

A responsive College Landing Page Website built using HTML, CSS, and JavaScript.
This project represents a modern landing page design for Lakshmi Narain College of Technology (LNCT), Bhopal.

🚀 Features

- Responsive Landing Page Design
- Modern UI Layout
- Courses Section
- Campus Information
- Facilities Section
- Student Testimonials
- Admission Call-to-Action Section
- Clean and Simple Navigation

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

📂 Project Structure

index.html
index.css
index-responsive.css
index.js
images/
